# chrome

chrome是google推出的开源的浏览器。国内很多浏览器都是基于chrome内核开发的，这里以windows系统介绍chrome浏览器的开发使用。

查看网页源代码

通过右键->查看网页源代码(ctrl+U)，查看其他网站源码有助于我们学习，可以帮助我们分析问题。

调试工具(devTools)

通过右键->检查(F12)打开控制台工具，devTools 是页面调试一定会使用的工具。

devTools 常用功能介绍：

  • Elements 元素：调试 HTML 页面，调试 DOM 和 CSS 等页面样式。
  • Network 网络：调试网络请求响应，了解页面资源加载速度，网页性能检测。
  • Console 控制台：调试 JS、查看 log 日志。
  • Sources 源代码资源：调试页面源代码，可加断点调试。
  • Application 应用：调试客户端存储，Cookie、Storage。

# vscode

Visual Studio Code(简称 vscode)是微软推出的运行于Mac OS X、Windows 和 Linux 之上的，针对于编写现代 Web 和云应用的跨平台源代码编辑器。

下载安装

vscode 官网 (opens new window)

  • 进入官网选择稳定的发行版本下载

  • 下载好后打开文件按引导安装(傻瓜式安装，同意下一步就行)

  • 安装好后默认语言是英文，在扩展(crtl+shift+x)中搜索Chinese下载中文语言包

  • 建议下载一个 live server(本地开发服务器)，能够自动替换更改内容

  • 下载好 live server 后，在 html 页面文件右键->open with live server启动本地开发服务

# 基础知识

HTML 不是一种编程语言，而是一种标记语言 (超文本标记语言)，是网页制作所必备的。用于按不同标签声明网页中的内容。

一个网页是由很多区域组合构成的，即使用 HTML 标签布局。

# 基本结构

  • DOCTYPE 声明为 HTML 文档
  • html lang:网页的语言，如 en/zh-CN 等，非必选项目
  • head 文档说明部分，对搜索引擎提供信息或加载 CSS、JS 等
  • body 页面主体内容
<!DOCTYPE html>
<html lang="zh-CN">
  <head>
    <meta charset="utf-8" />
    <meta name="keyword" content="Javascript,HTML,CSS" />
    <meta name="description" content="WEB应用程序开发开发" />
    <title>Web</title>
  </head>
  <body></body>
</html>

# 文件命名

  • 文件命名使用小写字母、下划线、中线组合命名文件，多个单词可以使用- 或 _ 连接，建议使用 - 字符如 user-create.html，不要出现中文字符。

  • 扩展名标准是.html，当然也可以使用.htm(最开始操作系统不支持四位扩展名)。

  • 在服务器中很多不是.html 或.htm 结尾的文件也能被编译成.html 文件，这是因为服务器使用了不同的编译模板，如 C#中的 razor 渲染引擎编译.cshtml 文件成标准的.html 文件。

    注：浏览器目前只能识别标准的.html 结尾的文件。

# 代码注释

使用注释对一段 html 代码进行说明，方便自己或同事在未来清楚的明白代码意图。

<!-- 这是导航条 START -->

  <nav>
    <ul>
      <li>
        <a href="">返回首页</a>
      </li>
      <li>
        <a href="">登录</a>
      </li>
    </ul>
  </nav>
</header>
<!-- 这是导航条 END -->
